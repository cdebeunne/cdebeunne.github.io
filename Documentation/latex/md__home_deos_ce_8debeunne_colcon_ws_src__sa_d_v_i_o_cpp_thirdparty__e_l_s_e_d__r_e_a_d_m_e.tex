This repository contains the source code of \href{https://doi.org/10.1016/j.patcog.2022.108619}{\texttt{ {\bfseries{E\+L\+S\+E\+D\+: Enhanced Line S\+Egment Drawing}}}} the fastest line segment detector in the literature. The code depends on Open\+CV (tested with version 4.\+1.\+1). $<$details$>$

In Ubuntu 18.\+04 compile it from sources with the following instructions\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\# Install dependencies (Ubuntu 18.04)}
\DoxyCodeLine{sudo apt-\/get install -\/y build-\/essential cmake git libgtk2.0-\/dev pkg-\/config libavcodec-\/dev libavformat-\/dev libswscale-\/dev}
\DoxyCodeLine{\# Download source code}
\DoxyCodeLine{git clone https://github.com/opencv/opencv.git -\/-\/branch 4.1.1 -\/-\/depth 1}
\DoxyCodeLine{\# Create build directory}
\DoxyCodeLine{cd opencv \&\& mkdir build \&\& cd build}
\DoxyCodeLine{\# Generate makefiles, compile and install}
\DoxyCodeLine{cmake -\/D CMAKE\_BUILD\_TYPE=RELEASE -\/D CMAKE\_INSTALL\_PREFIX=/usr/local ..}
\DoxyCodeLine{make -\/j}
\DoxyCodeLine{sudo make install}
\end{DoxyCode}
