\doxysubsection{Detailed Description}
A 2D line feature class. 

\mbox{\hyperlink{classisae_1_1_line2_d}{Line2D}} class represents a line feature in the image, labeled as \char`\"{}linexd\char`\"{}. It inherits from \mbox{\hyperlink{classisae_1_1_a_feature}{A\+Feature}} and can hold multiple 2D points representing the line.
