\doxysubsection{Detailed Description}
Angular error cost function for a point landmark in the sensor frame. Parameters are delta update of the frame pose and the landmark position. 

This cost function uses bearing vector that is compared to the projection of the landmark on the unit sphere, via an arbitrary tangent plane. This is inspired by \char`\"{}\+V\+I\+N\+S-\/\+Mono\+: A Robust and Versatile Monocular Visual-\/\+Inertial State
\+Estimator\char`\"{} by Qin et al. Source\+: \href{https://arxiv.org/abs/1708.03852}{\texttt{ https\+://arxiv.\+org/abs/1708.\+03852}}
