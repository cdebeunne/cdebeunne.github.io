\doxysubsection{Detailed Description}
The core abstract class of the S\+L\+AM system. It handles front-\/end, back-\/end and initialization. 

This is the skeleton of the S\+L\+AM system that will call all the subsystems (sensor processing, optimizer, map processing...) It loads all the parameters of the S\+L\+AM and implements front\+End and back\+Ends that must be called in a separated thread. A few important methods relative to feature processing and pose estimation are implemented as well to make the \mbox{\hyperlink{classisae_1_1_s_l_a_m_core_ab97c16d22ca0c3d50c4ef63bcd6c88f7}{front\+End\+Step()}} and \mbox{\hyperlink{classisae_1_1_s_l_a_m_core_aa378dd826e38a3bb5358a261466a97e3}{back\+End\+Step()}} methods readable. Also, a few profiling variables and methods are implemented to monitor the performances.
