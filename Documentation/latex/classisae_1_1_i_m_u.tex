\doxysubsection{Detailed Description}
An Inertial Measurement Unit (\mbox{\hyperlink{classisae_1_1_i_m_u}{I\+MU}}) sensor class. 

This class represents an \mbox{\hyperlink{classisae_1_1_i_m_u}{I\+MU}} sensor that provides acceleration and gyroscope measurements. It handles the preintegration of \mbox{\hyperlink{classisae_1_1_i_m_u}{I\+MU}} data for use in S\+L\+AM systems.
