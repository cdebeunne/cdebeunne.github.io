\doxysubsection{Detailed Description}
A 2D bounding box feature class. 

\mbox{\hyperlink{classisae_1_1_b_box2_d}{B\+Box2D}} class represents a bounding box feature in the image, labeled as \char`\"{}bboxxd\char`\"{}. It inherits from \mbox{\hyperlink{classisae_1_1_a_feature}{A\+Feature}} and can hold multiple 2D points representing the corners of the bounding box.
