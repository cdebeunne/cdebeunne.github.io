\doxysubsection{Detailed Description}
An image sensor class that uses the \mbox{\hyperlink{classisae_1_1_omni}{Omni}} model. 

The implementation of the \mbox{\hyperlink{classisae_1_1_omni}{Omni}} model is based on the paper\+: \char`\"{}\+Single View Point Omnidirectional Camera Calibration from Planar Grids\char`\"{} by Mei et al. Available at \href{https://www.robots.ox.ac.uk/~cmei/articles/single_viewpoint_calib_mei_07.pdf}{\texttt{ https\+://www.\+robots.\+ox.\+ac.\+uk/$\sim$cmei/articles/single\+\_\+viewpoint\+\_\+calib\+\_\+mei\+\_\+07.\+pdf}}
