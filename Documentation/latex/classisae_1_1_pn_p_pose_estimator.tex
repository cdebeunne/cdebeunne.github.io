\doxysubsection{Detailed Description}
\mbox{\hyperlink{classisae_1_1_pn_p_pose_estimator}{Pn\+P\+Pose\+Estimator}} class for estimating the transformation between two frames using PnP. 

The estimation is done by using the PnP algorithm on the matches between the two frames. The matches must be associated with 3D landmarks.
