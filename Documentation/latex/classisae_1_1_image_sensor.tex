\doxysubsection{Detailed Description}
Abstract class for image sensors. 

This class inherits from \mbox{\hyperlink{classisae_1_1_a_sensor}{A\+Sensor}} and provides additional functionalities specific to image sensors. It includes methods for handling depth data, image pyramids, feature extraction, and projection. It has also all the methods to apply image processing techniques such as C\+L\+A\+HE, histogram equalization etc...
