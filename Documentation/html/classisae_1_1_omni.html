Detailed Description

An image sensor class that uses the Omni model.

The implementation of the Omni model is based on the paper: "Single View Point Omnidirectional Camera Calibration from Planar Grids" by Mei et al. Available at https://www.robots.ox.ac.uk/~cmei/articles/single_viewpoint_calib_mei_07.pdf
